Object Detection System Overview

I built an object detection system using the Pascal VOC 2012 dataset from Kaggle. The model was trained using YOLOv5 with Kaggle's GPU acceleration to enhance performance and reduce training time.

After training, the model weights were exported from the .pt format to the more portable ONNX format to ensure broader compatibility across platforms and inference engines.

The trained model and outputs were downloaded locally, where OpenCV was used for real-time predictions on images, videos, and live webcam input.

For accessibility and user interaction, the detection system was integrated into a Streamlit web interface , allowing image uploads and real-time detection directly on mobile devices.
